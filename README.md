# Alternative Assets Market Tracker

An end-to-end Business Intelligence project simulating Preqin/PitchBook-style analytics for the alternative assets industry.

## 🚀 Features
- Python ETL pipeline (Ingest → Validate → Transform → Export)
- Data quality monitoring (completeness, duplicates, invalid values)
- Star schema (Firms, Funds, Sectors, Geography, Transactions, Outreach)
- BI-ready CSVs for Tableau
- Tableau Dashboards:
  - Market Overview
  - Data Quality Monitor
  - Firms & Funds Explorer
  - Transactions Analysis
  - Outreach Tracker
